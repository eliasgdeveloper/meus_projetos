Enunciado do Problema:
Você foi contratado por um supermercado para desenvolver uma função que calcula o valor total da compra de um cliente com base nos produtos adquiridos. Os preços dos produtos são fixos e são os seguintes:

Leite: R$ 3,50
Pão: R$ 1,75
Chocolate: R$ 5,25
O cliente fornece uma lista de produtos que deseja comprar e a função deve retornar o valor total da compra.

Requisitos:
Escreva uma função chamada calcularTotalCompra que recebe um array de strings chamado produtos.
A função deve calcular o total da compra somando os preços dos produtos listados no array.
Utilize uma estrutura de controle (loop) para iterar sobre os produtos e condicional (if-else) para adicionar o preço correto ao total para cada produto.
A função deve retornar o valor total da compra.
Exemplo de Uso:
Considere o array de produtos ["leite", "pao", "leite", "chocolate", "chocolate"].

Ao chamar a função calcularTotalCompra(produtos), ela deve retornar o valor 19.25.

Implemente a função e verifique o resultado com o exemplo fornecido.
